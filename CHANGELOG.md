## [0.1.0] - 2026-01-04

### Highlights

- Universal HTTP/HTTPS proxy with MITM support for traffic capture, database storage, and cross-platform system proxy configuration

### Added

- HTTP forward proxy with HTTPS MITM (Man-in-the-Middle) interception
- Reverse proxy with automatic TLS via Let's Encrypt/ACME
- Traffic capture in multiple formats: NDJSON, HAR, IR (APISpecRift)
- Database storage backends: SQLite (team mode) and PostgreSQL (production mode)
- Async traffic storage with configurable queue size, batch size, and workers
- Request filtering by host, path, and HTTP method with wildcard support
- Binary content detection using magic bytes and MIME type heuristics
- Proxy chaining through upstream proxy servers
- Prometheus metrics and health check endpoints (/metrics, /healthz, /readyz)
- System proxy configuration for macOS, Windows, and Linux (GNOME/KDE)
- Pure Go CA certificate generation (no OpenSSL dependency)
- CA certificate installation into system trust stores
- YAML configuration file support
- CLI commands: serve, reverse, daemon, ca, system, config
- Docker support with multi-stage Dockerfile and docker-compose
- Sensitive header filtering (Authorization, Cookie, etc.) by default

### Documentation

- Comprehensive README with usage examples for all deployment modes
- Database URL format documentation for SQLite and PostgreSQL
- Prometheus metrics reference table
- Configuration file schema with examples

### Build

- GitHub Actions CI/CD workflows for build, lint, and test
- GoReleaser configuration for multi-platform binary releases
- Dependabot configuration for Go modules and GitHub Actions
